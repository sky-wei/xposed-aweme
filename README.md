# xposed-aweme

### 基于Xposed框架写的 `抖音` 插件(已停止维护)

#### 主要的功能
1. 自动切换下一个视频播放(用户可自定义时间)
2. 自动关注与点赞(切换视频时触发)
3. 手动或自动下载无水印视频到本地(切换视频时触发)
4. 自动评论(无需打开讨论界面即可完成)
5. 解除视频15秒限制(用户可自定义时间)

#### 配置的入口
1. 视频界面 -> 分享 -> 抖音助手

##### 支持的版本
1. v2.1.4版本支持抖音(v2.0.0,v2.0.1,v2.1.0,v2.2.1,v2.3.0,v2.4.0,v2.5.0,v2.6.0)
